# Prompts

Saved prompts and LLM sessions **about building the guide** — the meta layer. Things like: a prompt used
to audit the pack for contradictions, a session that restructured the documents, a reusable brief for
reviewing prose against the house style.

These are working notes. They are out-of-scope for the book, absent from the `pack` branch, and nothing
here is shipped to readers.

## What does *not* belong here

**`src/pack/story-continuity-checker-prompt.md` is not one of these.** It is a prompt, but it is
*product*: a tool the guide hands to creators to run against their own node's canon. It is item 10 in
the book's reading order and ships in every output.

The test is the audience, not the file type:

- A prompt **the reader is given** → `src/pack/`.
- A prompt **we used to make the book** → here.
